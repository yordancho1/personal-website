Project Overview

In Summer 2025, I completed a comprehensive freelance web development project for Sistemi Nikolov EOOD, a professional roofing company based in Ruse, Bulgaria. This project showcased my ability to manage the entire web development lifecycle from initial client consultation to final deployment.

The client needed a modern, professional online presence to showcase their roofing services, including roof repairs, seamless gutters (5" and 6"), and custom metal sheet fabrication up to 7 meters using CNC technology. The website needed to be fully responsive, user-friendly, and optimized for local search visibility.

Challenges & Solutions

Challenge: Creating a professional, trustworthy design for a traditional industry (roofing) while maintaining modern web standards and user experience.

Solution: Balanced classic professional aesthetics with contemporary design elements. Used clean layouts, professional typography, and strategic use of imagery to build credibility while keeping the interface modern and accessible.

Challenge: Ensuring the website works perfectly in Bulgarian language with proper character encoding and text rendering.

Solution: Implemented proper UTF-8 encoding and tested extensively with Cyrillic characters to ensure all text displays correctly across all browsers and devices.

Challenge: Managing client expectations and timelines while delivering a high-quality product.

Solution: Maintained clear communication throughout the project, provided regular updates and previews, and set realistic milestones that were consistently met.

  • Apply technical skills in HTML, CSS, JavaScript to create a production-ready website
  • Handle web infrastructure including hosting setup and domain configuration
  • Deliver professional results independently as a freelance developer

The website continues to serve as the company's primary digital presence, helping them reach new clients and showcase their professional roofing services online.
